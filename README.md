Project Overview

The Student Management System is a console-based Python application developed using Object-Oriented Programming (OOP) concepts.
It allows users to manage student records efficiently by performing operations such as adding, viewing, searching, updating, and deleting student information.
The system also supports exporting student records to a CSV file for reporting purposes.

Features
Add new student records
View all students
Search students by Student ID
Update student marks
Delete student records
Calculate average marks
Export student reports to CSV format
